# OpenCode Project

This repository contains a collection of specialized AI agents designed to assist with various software development tasks.

## Setup Instructions

To set up this project, follow these steps:

1. Clone the repository to your home directory:

   ```bash
   cd ~ && git clone <repository-url>
   ```

2. Navigate to the project directory and run the setup task:
   ```bash
   cd opencode && task setup
   ```

## Agent Overview

This project includes several specialized agents, each designed for specific development tasks:

- **Claude-md-compliance-checker**: Verifies that code changes adhere to project-specific CLAUDE.md guidelines
- **Cloud-architect**: Assists with cloud architecture design and deployment strategies
- **Code-quality-pragmatist**: Reviews code for over-engineering and unnecessary complexity
- **Code-summarizer**: Specializes in analyzing, summarizing, and explaining existing code with step-by-step analysis
- **Conventional-commit-generator**: Specializes in generating conventional commits with concise explanations of changes made
- **Developer**: General software development assistance and code generation
- **Documentation**: Writes and maintains project documentation
- **Go-architecture-expert**: Provides Go-specific architecture design guidance
- **Go-code-generator**: Generates complete, production-ready Go code from specifications
- **Jenny**: Verifies that implemented solutions match project specifications
- **Karen**: Assesses actual project completion state and creates realistic finish plans
- **Legacy-upgrader**: Specializes in upgrading legacy codebases to modern versions and coding standards
- **Operations**: Handles operations tasks, deployment, and infrastructure questions
- **Performance**: Assists with performance analysis and optimization strategies
- **Reviewer**: Reviews code for best practices and potential improvements
- **Security**: Performs security analysis and vulnerability identification
- **Senior-typo3-developer**: Specializes in TYPO3 development with deep expertise in CMS architecture, extensions, and best practices
- **Software-architect**: Provides software architecture design and system design patterns
- **Task-completion-validator**: Validates that implemented features actually work as claimed
- **Technical-debt-analyzer**: Specializes in identifying, categorizing, and providing remediation strategies for technical debt
- **Tester**: Assists with test development and testing strategies
- **UI-comprehensive-tester**: Performs thorough UI testing of web and mobile applications
- **Ultrathink-debugger**: Diagnoses complex bugs, errors, and system failures

Each agent serves a specific purpose in the software development lifecycle, from code generation and testing to architecture design and security analysis.
